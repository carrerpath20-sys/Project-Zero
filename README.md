# 🔥 Zero Recon Framework — স্বয়ংক্রিয় রেড-টিম রিকনেসান্স  
warning:(Private project not for public use )

**Zero Recon** হলো একটি **এআই-চালিত, মডুলার রিকনেসান্স ফ্রেমওয়ার্ক**, যা বাগ বাউন্টি হান্টার এবং রেড টিমারদের জন্য ডিজাইন করা হয়েছে। এটি ARTEMIS-স্টাইল মাল্টি-এজেন্ট আর্কিটেকচার ব্যবহার করে, যা Windows-এ নেটিভভাবে চলে এবং WAF/ফায়ারওয়াল বাইপাস করতে সক্ষম।

---

## ⚡ **কেন এটি ব্যবহার করবেন?**

- 🧠 **এআই-চালিত অর্কেস্ট্রেশন**: কাজের জটিলতা বুঝে Cerebras / OpenRouter দিয়ে স্বয়ংক্রিয়ভাবে মডেল সিলেক্ট করে।
- 🛡️ **স্টেলথ ও ওপসেক (OPSEC)**: র্যান্ডম ইউজার-এজেন্ট, ডিলে, এবং প্রোক্সি সাপোর্ট।
- 🏗️ **১৫টি অ্যাডভান্সড রিকন ফেজ** (নিচে বিস্তারিত)।
- 🩹 **সেলফ-হিলিং (Self-Healing)**: কোনো ফেজ ব্যর্থ হলে AI নিজেই কোড/স্ট্র্যাটেজি রি-রাইট করে।
- 🔄 **অটো-আপডেট**: GitHub রিপো থেকে নতুন ভার্সন নিজে থেকেই টেনে আনে।
- 💻 **উইন্ডোজ-নেটিভ**: কোনো WSL বা লিনাক্স ভার্চুয়াল মেশিনের প্রয়োজন নেই।
